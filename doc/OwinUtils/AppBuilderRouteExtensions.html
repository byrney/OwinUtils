Branch Method

Creates a branch in the pipeline. If the inbound request matches "template" then the midlewares defined by branchBuilder will be called. Any parameters extracted by the template will be added to RouteParams for use within the branch. The BasePath and Path variables in the owin Environment will be adjusted and then restored when the branch is complete If the template is not matched this does nothing.
